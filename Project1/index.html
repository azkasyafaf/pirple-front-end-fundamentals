HTML Reference

In this project, I was tasked to search for 5 additional tags that haven't covered by the course.
Here's my search result complete with the explanation and reference.

  1. <abbr>
    The <abbr> tag is used to define an abbreviation or an acronym, for the example click here.

  2. <b>
    The <b> tag specifies bold text without any extra importance. Click here for example.

  3. <br>
    The <br> tag insert a single line break. It is usefull for writing poems. It is an empty tag which means it has no end tag. See here for the example.

  4. <em>
    The <em> tag is used to define emphasized text. The content inside is typically displayed in italic. Here is the example.

  5. <q>
    The <q> tag defines a short quotation. The content inside will quotation marks in front and end of the content. Click here for the example.
